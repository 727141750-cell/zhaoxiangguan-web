## 截图制作方法

### 方法1：模拟器截图
```bash
# 1. 启动模拟器
flutter emulators --launch Pixel_6

# 2. 运行应用
flutter run

# 3. 使用模拟器截图功能
# Android Studio: Tools -> Screenshots
# 或命令行: adb shell screencap -p /sdcard/screen.png
```

### 方法2：真实设备截图
1. 在真机上安装APK
2. 使用设备自带截图功能
3. 导出截图到电脑

### 方法3：设计制作（推荐）
使用设计工具(Figma/Sketch)制作：
1. 获取应用界面截图
2. 添加合适的背景
3. 添加文字标注
4. 调整尺寸和比例
5. 导出为PNG/JPG

## 设计建议

### 视觉风格
- **配色**: 与应用主题一致（蓝色系）
- **字体**: 清晰易读，中文使用系统字体
- **布局**: 简洁明了，重点突出
- **标注**: 使用半透明背景+白色文字

### 最佳实践
1. **展示真实功能** - 不要过度美化
2. **突出核心价值** - AI生成、多风格、高质量
3. **使用真实内容** - 真实的生成结果
4. **保持一致性** - 所有截图风格统一
5. **适配不同尺寸** - 确保文字在小屏幕清晰

### 应避免的问题
- ❌ 截图包含其他应用或系统元素
- ❌ 过度夸张的宣传语
- ❌ 使用非真实生成的图片
- ❌ 文字过多或过小
- ❌ 质量模糊的图片

## 🎨 截图规格要求

### Google Play 要求：
- **尺寸**：至少 320px 宽，最大 3840px 宽
- **长宽比**：16:9 或其他标准比例
- **格式**：JPG 或 24位 PNG（无阿尔法通道）
- **文件大小**：最大 8MB

### 推荐尺寸：
- **手机**：1080 x 1920（竖屏）
- **平板**：1920 x 1080（横屏）

## 📱 获取截图的方法

### 方法1：使用Android模拟器（推荐）

由于本地构建有问题，建议使用在线构建后的APK：

1. **构建APK**
   - 使用GitHub Actions自动构建
   - 或等待网络条件改善后手动构建

2. **安装到模拟器**
   ```bash
   # 安装APK到模拟器
   adb install app-release.apk

   # 或使用Flutter安装
   flutter install
   ```

3. **打开应用并截图**
   - 在模拟器中打开应用
   - 导航到各个页面
   - 使用Android Studio的截图工具：
     - View → Tool Windows → Logcat
     - 点击截图图标（相机图标）
     - 保存截图

### 方法2：使用真机截图

1. **构建并安装APK到手机**
2. **在手机上操作应用**
3. **使用手机截图功能**
   - 通常按：电源键 + 音量下键
4. **从手机导出截图**
   - 通过USB连接到电脑
   - 或使用云同步

### 方法3：暂时使用设计图

如果暂时无法获取真实截图，可以：
- 使用Figma/Sketch创建设计稿
- 使用在线工具制作应用预览图
- 参考现有类似应用的截图风格
